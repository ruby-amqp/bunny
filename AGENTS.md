# Instructions for AI Agents

## Overview

This is Bunny, a Ruby AMQP 0-9-1 client for RabbitMQ that has one
key dependency: [amq-protocol](https://github.com/ruby-amqp/amq-protocol),
an AMQP 0-9-1 protocol parser (more specifically: a serialization, deserialization, and framing implementation).

## Target Ruby Version

This library targets Ruby 3.0 and later versions.
